# Brief Gestion de stock G1 dans la plateforme sol
programme de gestion de stock avec mini dash board.
Ce programme a ete ecrit en python et la persistance des donnees avec mysql

## Les principales fonctionnlites 
- l'ajout de categorie
- L'affichage de la liste des categories
- La modiffication d'une categorie
- La suppression de categorie
- l'ajout de produit
- L'affichage de la liste des produits
- La modiffication du stock d'un produit
- La suppression de produit
- Effectuer une vente
- L'affichage de la liste des ventes
- afficher la valeur totale financiere
- afficher le produit le plus cher
- Afficher le nombre de produit par categorie
- un menu qui boucle

## Technologies Utilisees
- python3
- base de donnees mysql pour le stockage 
- programmation procedurale

### Configuration de la base de donnees
- installation de mysql 8 
- creation de la bse de donnees (avec le dump fourni)
- Configuration de la connexion a mysql en renseignant votre nom d'utilisateur et votre mot de passe
